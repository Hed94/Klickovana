# Klickovana
Primitivní webohá hra napsaná v JavaScriptu.

Cílem hry je co nejdéle utíkat se svým žlutým čtvercem uprostřed ostatním červeným objektům.
Můj osobní rekord je 28 sekund.

Hra vychází z původní klasiky dostupné na adrese
http://www.boskowan.com/www/jirka/zzz/klickovana.htm

Oproti původní verzi jsem celý kod přepsal a optimalizoval tak aby byla hra plynulejší a snadno se tam dají přidávat nebo ubírat další nepřátelé. Dále jsem změnil ovládání z myši na klávesnici a kvůli tomu jsem zcela odstranil černou oblast, neboť klávesnice je pohodlnější na ovládání, ale horší na zaměřování. A mimo to bylo přidáno také trackování nejvyšší dosažené hodnoty.
